# SP0256-AL2 Example Words

These are hand-entered pronunciation examples for the SP0256-AL2 allophone set.
They are useful starting points for the serial examples' raw allophone mode:

```text
:HH1 EH LL AX OW PA5
```

The examples below are not a complete text-to-speech dictionary. Expect to tune
pauses and repeated consonants by ear.

Datasheet examples often write the sibilant allophone as `SS`, and the two D
allophones as `DD1`/`DD2`. This repository uses `S`, `D1`, and `D2` so raw
serial input matches the sketch parser.

## Quick Test Lines

Paste these into the serial examples with the leading colon:

```text
:HH1 EH LL AX OW PA5
:RR1 OW PA2 BB2 AA PA3 TT2 PA5
:S S PA3 TT1 AA AA PA3 PP PA5
:YY2 EH EH S S PA4 NN2 OW PA5
:TT2 WH EH EH NN1 PA2 PA3 TT2 IY PA5 S S IH PA3 KK2 S PA5
```
